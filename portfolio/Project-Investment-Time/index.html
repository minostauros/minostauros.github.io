SwallowIT Blog

이 홈페이지 역시 프로젝트의 일환으로 제작되었습니다. 이에 따라 다른 기존에 만들어진 웹사이트와 비교해보았습니다. 비교 대상은 블로그 SwallowIT(이하 SIT)입니다.

  1. 네비게이션 영역

    SIT의 경우 네비게이션 영역이 화면 우측에 위치합니다. 로고, 공지사항, 카테고리 등이 나열되어 있고, 자주 사용하지 않는 메뉴는 기본적으로 숨겨져 있습니다. 사이드바를 마우스 드래그하여 움직일 수 있으나, 사이트 이동시 유지되지 않고 모바일에서 지원되지 않아 큰 쓸모는 없습니다.

    이 사이트는 네비게이션이 고정되어 있기 때문에, 어느 화면에서든지 네비게이션에 빠르게 접근 가능하며, 세로로 길게 늘어져 있는 컨텐츠라도 네비게이션에 있는 internel links를 이용하여 빠르게 이동할 수 있습니다.

  2. 반응형 웹 지원

    SIT는 화면 크기, 혹은 디바이스 종류에 따른 스타일시트를 구분하지 않고 있습니다.와이드 구성으로 이루어져 있어 사진을 보는 데는 불편함이 없으나, 스마트폰과 같은 작은 화면에서는 글씨가 매우 작아 매번 확대해야 하는 불편함이 있습니다. 또한, 본문 넓이도 고정 폭으로, 웹브라우저 가로 길이를 일정 px이하로 줄이면 컨텐츠가 잘려 보이게 되고 스크롤을 이용해야 합니다.

    반면 이 사이트는 어느 크기의 화면에서든지 왠만해서는 가로 스크롤바가 생기지 않습니다. 화면 가로폭에 각각 대응하여 3가지의 스타일시트로 구성되어 있으며, 각각의 화면에서 남는 공간 없이 최대한의 폭을 사용함으로써 컨텐츠의 가독성을 극대화시켰습니다. 특히 네비게이션바 같은 경우 제일 작은 폭에서는 아예 숨겨져 있다가 '≡' 버튼을 누르면 확장되는 방식으로 공간 효율성을 극대화 시킴과 동시에, 일반적인 스마트폰 앱들과 비슷한 사용 방법으로, 반응형 웹의 단점으로 꼽히는 '화면 크기가 달라질 때마다 새로운 사용 방식을 습득해야 한다'는 점을 극복했습니다.

  3. HTML5, CSS3

    SIT도 나름 유려한 외관을 갖추고 있지만, HTML이나 CSS의 최신기술을 활용하지 않았기 때문에, 자바스크립트 의존도가 높은 편입니다.

    이 사이트는 아직 안정화되진 않았지만 대부분의 모던 웹 브라우저에서 지원하는 각종 신기술 등을 적극 활용하여, 사이트의 외관과 최적화에 활용하고 있습니다. 칼럼형 구성과 박스 혹은 카드형의 디자인은 최신 트렌드를 따르면서도 자바스크립트 없이 구현이 가능함을 보여줍니다.

  4. Font

    SIT는 로컬에 지정된 폰트가 존재하지 않을 경우, 로컬에 있는 지정된 font-family로 폰트를 표시합니다. 컨텐츠가 원래 목적과는 다르게 표시될 가능성이 있습니다.

    현 사이트는 로컬에 지정된 폰트가 존재하지 않을 경우 폰트 파일을 인터넷으로부터 불러오는 방식을 채택했습니다. 인터넷 속도가 빠른 현 시대 상황을 고려했을 때 페이지 로딩에 큰 영향을 미치지 않기 때문에 오히려 외관적으로 이득을 볼 수 있습니다.

  5. 플래시의 사용

    SIT는 본문 제목의 그림자와 반영 효과를 나타내기 위해 플래시를 사용하고 있어, 모바일 브라우저 등에서의 확장성이 떨어집니다.

    요즘은 대부분의 인터넷에 접속 가능한 기기들이 HTML5와 CSS3를 지원하기 떄문에, 이를 적극 활용하여 플래시 없이도 완전한 효과 구현에 성공하였습니다. 아직 정착된 기술은 아니지만 blur filter등도 적극 활용하여 모바일 환경을 구축했습니다.

  6. 기타 웹 사이트 소개

    본 사이트는 정해진 특별한 주제 없이 개인적인 생각의 단편들을 의식의 흐름 기법으로 표현한 블로그입니다. 디자인은 화이트와 라이트 그레이가 주를 이루며, 포인트 칼라로 조금 진한 블루(#3472b1)를 사용하고 있습니다. 모던하면서도 각지지 않은 분위기를 위해 각진 폰트를 사용하면서도 둥근 모서리와 그림자를 적절히 사용했습니다. 메인 페이지 상단에는 공백 없이 꽃 이미지를 배치하여 신선함을 주고, 페이지 맨 하단에는 gradient효과를 활용하여 페이지가 끝나는 느낌을 적절히 표현했습니다.

    사이드 패널(혹은 header)는 어느 페이지에서든 정확한 곳으로 이동할 수 있도록 늘 같은 위치에서 링크를 제공합니다. 패널 상단의 아바타(얼굴 이미지)와 웹 사이트 제목을 클릭하면 root로 이동하며, 나머지 항목들은 페이지 내에서 쉽게 이동할 수 있도록 internel link들을 배치했습니다. 모바일에서는 스마트폰과 같은 직관적은 UI로 변신하여, 좌측 상단에 HOME버튼을, 우측 상단에 메뉴를 여닫을 수 있는 버튼을 제시하고 있습니다.
